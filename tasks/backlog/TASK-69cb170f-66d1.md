# Create first product

## Objective
Set up the first product directory in the monorepo with its own CLAUDE.md, language-specific configuration, and source structure. This establishes the pattern all future products will follow.

## Scope
### Owns (files this task may create/modify)
- `products/{product-name}/`
- `products/{product-name}/CLAUDE.md`
- `products/{product-name}/src/` (or language-appropriate equivalent)
- `context/state.md` (update current focus)

### Reads (files this task depends on but must NOT modify)
- Root `CLAUDE.md`
- `context/decisions.md`

## Contracts
### Consumes
- The task tracking system from [[TASK-69cb170c-0615]]

### Publishes
- A product directory template that future products will follow
- Product-level CLAUDE.md conventions

## Acceptance Criteria
### Automated (must pass, machine-verifiable)
- [ ] `products/{name}/` directory exists with expected structure
- [ ] Product-level CLAUDE.md exists and is under 100 lines
- [ ] At least one source file or placeholder exists

### Judgment (requires review)
- [ ] Directory structure is language-appropriate
- [ ] Product CLAUDE.md covers: build commands, test commands, code style
- [ ] Pattern is general enough for other products to follow

- Product and language choice TBD (to be decided in [[TASK-69cb170d-6830]] or by user)

## Context
This task depends on the bootstrap being complete. The specific product and language will be determined either by the roadmap task or by user direction.
